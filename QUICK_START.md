Ваш проект содержит **3 готовых приложения** для ray tracing в ASCII:

## 📺 Приложение 1: GUI (Рекомендуется) ⭐

**Полнофункциональное окно с меню конфигурации**

```powershell

```

✅ **Что вы получите:**
- 🖼️ Красивое окно приложения

## ⌨️ Управление (одинаковое для всех)

### Движение
| Клавиша | Действие |
|---------|----------|
| **W** | Вперёд |
| **S** | Назад |
| **A** | Влево |
| **D** | Вправо |
| **Q** | Вниз |
| **E** | Вверх |

### Камера
| Клавиша | Действие |
|---------|----------|
| **↑↓←→** | Повернуть |
| **R** | Сброс |
| **ESC** | Выход |

## 🚀 Рекомендуемый первый запуск

1. **Запустите GUI приложение** (самое красивое):
   ```powershell
   .\run_gui.bat
   ```

2. **Откройте конфиг меню** (File → Configuration)

3. **Попробуйте изменить параметры:**
   - Разрешение (Resolution dropdown)
   - Размер символов (Character Size dropdown)
   - Скорость анимации (Animation Speed slider)
   - Автоповорот (Auto Rotate checkbox)

4. **Используйте WASD** для управления камерой в реальном времени

5. **Нажмите ESC** для выхода (конфиг сохранится автоматически)

## ❓ FAQ

**Q: Какое приложение мне запустить?**
A: Если сомневаетесь - запустите GUI (`.\run_gui.bat`)

**Q: Как сохранить конфиг?**
A: Автоматически при выходе из GUI приложения

**Q: Как вернуть настройки по умолчанию?**
A: Удалите `config.ini` и перезапустите

**Q: Почему низкий FPS?**
A: Снизьте разрешение или размер символов

**Q: Где находится исходный код?**
A: `src/` папка. Главные приложения:
- `src/apps/gui/main.cpp` - GUI
- `src/apps/interactive/main.cpp` - Console
- `src/apps/viewer/main.cpp` - Batch
