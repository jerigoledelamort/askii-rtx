# 🖼️ ASCII RTX v1 - GUI Application with ImGui

Полнофункциональное графическое приложение для интерактивной визуализации ASCII ray tracing с встроенным меню конфигурации.

## 🚀 Запуск

### Вариант 1: Через .bat файл (рекомендуется)
```bash
.\run_gui.bat
```

### Вариант 2: Через PowerShell
```powershell
.\build\ascii_rtx_gui.exe
```

### Вариант 3: Через CMake
```powershell
cmake --build .\build --config Release
.\build\ascii_rtx_gui.exe
```

## 🎮 Интерфейс и управление

### Главное меню (верхняя панель)
- **File** - Сохранение конфигурации и выход
- **View** - Управление видимостью окон и статистики
- **Help** - Справка по управлению

### Окно конфигурации (Configuration Window)
Основное окно с  настройками приложения. Разделено на разделы:

#### 📺 Video Settings
- **Resolution** - Выбор разрешения (dropdown):
  - 800x600
  - 1024x768
  - 1280x720 (рекомендуется)
  - 1600x900
  - 1920x1080
  - 2560x1440

- **Fullscreen** - Переключатель полноэкранного режима
- **V-Sync** - Вертикальная синхронизация

#### 🔤 ASCII Settings
- **Character Size** - Размер символа (dropdown):
  - 8x16 (компактный)
  - 10x20
  - 12x24 (по умолчанию)
  - 14x28
  - 16x32 (крупный)

- **ASCII Grid** - Отображение текущего размера сетки

#### 📷 Camera Settings
- **Move Speed** - Скорость движения камеры (0.01 - 1.0)
- **Rotation Speed** - Скорость вращения (0.01 - 0.5)
- **Auto Rotate** - Автоматическое вращение сцены
- **Reset Camera** - Кнопка сброса позиции камеры

#### 🎨 Scene Settings
- **Animation Speed** - Скорость анимации объектов (0.0 - 3.0)
- **Number of Objects** - Количество объектов в сцене (1 - 10)

### ⌨️ Управление клавиатурой

#### Движение камеры
| Клавиша | Действие |
|---------|----------|
| **W** | Вперёд |
| **S** | Назад |
| **A** | Влево |
| **D** | Вправо |
| **Q** | Вниз |
| **E** | Вверх |

#### Камера
| Клавиша | Действие |
|---------|----------|
| **↑** | Вверх (Pitch) |
| **↓** | Вниз (Pitch) |
| **←** | Влево (Yaw) |
| **→** | Вправо (Yaw) |

#### Горячие клавиши
| Клавиша | Действие |
|---------|----------|
| **R** | Сброс позиции камеры |
| **ESC** | Выход из приложения |

### 📊 Статистика

Окно "Statistics" показывает в реальном времени:
- **FPS** - Текущая частота кадров
- **Frame Time** - Время рендеринга одного кадра в миллисекундах
- **Resolution** - Текущее разрешение окна
- **ASCII Grid** - Размер ASCII сетки

### Ключевые классы

#### `AsciiRtxGuiApp`
Главный класс приложения:
- Создание и управление окном
- Инициализация Direct3D 11
- Главный цикл событий
- Обработка клавиатуры

#### `AppConfig`
Конфигурация приложения:
- Сохранение/загрузка параметров в `config.ini`
- Управление видео параметрами
- Управление параметрами камеры
- Управление ASCII параметрами

#### `AsciiRenderer`
Рендеринг ASCII сцены:
- Симуляция 3D сцены через градиенты
- Конвертация RGB в ASCII символы
- Буферизация пиксельных данных

#### `ImGuiMenu`
UI меню:
- Главное меню (File, View, Help)
- Окно конфигурации
- Окно статистики

## 💾 Конфигурация

Все параметры автоматически сохраняются в файл `config.ini` при закрытии приложения и загружаются при запуске.

### Формат config.ini

```ini
; ASCII RTX v1 - GUI Application Configuration

[VIDEO]
window_width=1280
window_height=720
fullscreen=false
vsync=true

[ASCII]
char_width=12
char_height=24

[CAMERA]
camera_speed=0.1
auto_rotate=true

[SCENE]
animation_speed=1.0

[UI]
show_stats=true
```

## 🎨 Кастомизация

### Изменение цветовой схемы

В `imgui_setup.cpp` функции `setup_style()`:

```cpp
void ImGuiSetup::setup_style() {
	// Выберите одну из:
	ImGui::StyleColorsDark();       // Тёмная тема
	ImGui::StyleColorsLight();      // Светлая тема
	ImGui::StyleColorsClassic();    // Классическая

	// Или кастомизируйте вручную
	ImGuiStyle& style = ImGui::GetStyle();
	style.Colors[ImGuiCol_WindowBg] = ImVec4(...);
}
```

### Добавление новых опций

1. Добавьте переменную в `AppConfig` структуру
2. Добавьте сохранение/загрузку в `app_config.cpp`
3. Добавьте UI элемент в `draw_config_window()` в `imgui_setup.cpp`

Пример:

```cpp
// app_config.h
struct AppConfig {
	float my_param = 0.5f;
};

// imgui_setup.cpp
if (ImGui::CollapsingHeader("My Settings")) {
	ImGui::SliderFloat("My Param##slider", &config_.my_param, 0.0f, 1.0f);
}
```

## 🐛 Возможные проблемы

### Приложение не запускается
- Убедитесь, что проект скомпилирован: `cmake --build .\build --config Release`
- Проверьте наличие файла `build\ascii_rtx_gui.exe`

### Окно не отображается
- Может быть создано в фоне
- Попробуйте Alt+Tab для переключения окон
- Переезагрузитесь и попробуйте снова

### Низкий FPS
- Уменьшите разрешение
- Уменьшите размер ASCII символов
- Отключите автоматическое вращение
- Уменьшите анимационную скорость

### Меню не реагирует
- Убедитесь, что окно приложения активно
- Нажмите на него
