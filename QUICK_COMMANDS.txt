🔧 ОСНОВНЫЕ КОМАНДЫ
====================

[ПЕРВЫЙ РАЗ] Скачать зависимости:
   .\download_deps.ps1

[ПОЛНАЯ СБОРКА] Собрать с нуля:
   build.bat all

[ПЕРЕСБОРКА] После редактирования:
   cd build
   cmake --build . --config Release --parallel 4
   cd ..
   .\build\ascii_rtx_viewer.exe

[ЧИСТАЯ СБОРКА] Удалить и пересобрать:
   Remove-Item -Recurse -Force build
   build.bat all


📋 РЕДАКТИРОВАНИЕ СЦЕНЫ
=======================

1. Открыть файл конфигурации:
   src\config\default.cpp

2. Найти функцию:
   void init_default_config()

3. Добавить объект (пример - зеркальная сфера):
   g_config.scene.objects[4].type = 0;           // 0=sphere, 1=box, 2=plane
   g_config.scene.objects[4].px = 1.5f;          // X позиция
   g_config.scene.objects[4].py = 0.5f;          // Y позиция
   g_config.scene.objects[4].pz = 0.0f;          // Z позиция
   g_config.scene.objects[4].sx = 0.3f;          // Размер X
   g_config.scene.objects[4].sy = 0.3f;          // Размер Y
   g_config.scene.objects[4].sz = 0.3f;          // Размер Z
   g_config.scene.objects[4].material_id = 4;    // Mirror материал
   g_config.scene.num_objects = 5;               // Увеличить счётчик

4. Пересобрать:
   cd build
   cmake --build . --config Release
   cd ..

5. Запустить:
   .\build\ascii_rtx_viewer.exe


⚙️ ПАРАМЕТРЫ КОНФИГУРАЦИИ
=========================

В default.cpp можно изменять:

Разрешение и FPS:
   g_config.window.width = 1920;
   g_config.window.height = 1080;
   g_config.window.fps = 60;

Качество рендеринга:
   g_config.render.samples = 2;        // Сэмплы на пиксель (1-8)
   g_config.render.exposure = 1.0f;    // Яркость (0.5-2.0)
   g_config.render.gamma = 1.0f;       // Гамма (0.8-2.2)

Камера:
   g_config.camera.radius = 2.0f;      // Расстояние
   g_config.camera.height = 1.0f;      // Высота
   g_config.camera.fov = 45.0f;        // Поле зрения

Освещение:
   g_config.light.intensity = 1.0f;    // Интенсивность
   g_config.light.x = 1.0f;
   g_config.light.y = 1.0f;
   g_config.light.z = 1.0f;


🎨 МАТЕРИАЛЫ (material_id)
==========================

0 = Matte           (матовая)
1 = Glossy Plastic  (пластик)
2 = Mixed Material  (смешанный)
3 = Glass           (стекло)
4 = Mirror          (зеркало) ← красиво!
5 = Metal           (металл)
6 = Blue Diffuse    (синяя)
7 = Emissive        (светящийся)
