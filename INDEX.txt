🛠️ СБОРКА И ЗАПУСК
==================

Windows:
--------
# Вариант 1: Автоматический batch
build.bat all

# Вариант 2: PowerShell
.\setup.ps1 -FullBuild

# Вариант 3: Ручная сборка
mkdir build && cd build
cmake -G Ninja -DCMAKE_BUILD_TYPE=Release ..
cmake --build . --config Release
cd ..
.\build\ascii_rtx_viewer.exe

Linux/macOS:
-----------
mkdir build && cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
cmake --build . --parallel 4
./ascii_rtx_viewer

🎯 ЧТО МОЖНО СДЕЛАТЬ ДАЛЬШЕ
===========================

Редактирование сцены:
1. Отредактировать src/config/default.cpp
2. Добавить объекты или изменить параметры
3. Пересобрать: cmake --build ./build
4. Запустить

Добавление нового материала:
1. Добавить в materials.cpp
2. Использовать material_id в объектах
3. Пересобрать

Оптимизация:
1. Уменьшить render.samples (1-2)
2. Уменьшить разрешение (640x480)
3. Увеличить параметры для финального рендера
