# 🎊 ASCII RTX v1 - ПОЛНЫЙ ПРОЕКТ ЗАВЕРШЁН! 🎉

## 📊 Итоги проекта

Ваш **ASCII Ray Tracing Engine** теперь включает **3 полнофункциональных приложения** с поддержкой управления, конфигурации и интерактивной визуализации!

---

## 🚀 Три готовых приложения

### 1️⃣ **GUI Приложение** ⭐ (РЕКОМЕНДУЕТСЯ)
```powershell
.\run_gui.bat
```
- ✨ DirectX 11 + ImGui
- 🎛️ Выпадающие меню для конфигурации
- 📺 Выбор разрешения (800x600 - 2560x1440)
- 🔤 Выбор размера символов (8x16 - 16x32)
- 💾 Автосохранение в config.ini
- ⌨️ WASD управление + стрелки

### 2️⃣ **Interactive Console**
```powershell
.\build\ascii_rtx_interactive.exe
```
- 🎮 Минималистичная консоль
- ⌨️ Реальное время WASD управление
- 📊 FPS счётчик и координаты
- ~60 FPS стабильно

### 3️⃣ **Batch Renderer**
```powershell
.\run.bat
```
- 📈 500 кадров с полной статистикой
- 📊 Анализ производительности
- 🤖 Подходит для CI/CD

## 🔐 Сохранение конфигурации

### Где хранится
- **Файл:** `config.ini` (в директории проекта)
- **Формат:** INI (текстовый)
- **Обновляется:** при закрытии GUI приложения

### Что сохраняется
```ini
[VIDEO]
window_width=1280
window_height=720
fullscreen=false
vsync=true

[ASCII]
char_width=12
char_height=24

[CAMERA]
camera_speed=0.1
auto_rotate=true

[SCENE]
animation_speed=1.0
```

### Как сбросить
```powershell
# Просто удалите файл
Remove-Item config.ini
# Перезапустите - загрузятся параметры по умолчанию
```
