## ⌨️ Управление

### Движение
| Клавиша | Действие |
|---------|----------|
| **W/A/S/D** | Вперёд/Влево/Назад/Вправо |
| **Q/E** | Вниз/Вверх |
| **↑↓←→** | Повернуть |

### Горячие клавиши
- **R** - Сброс камеры
- **ESC** - Выход

### Меню (только GUI)
- **File** - Сохранение и выход
- **View** - Управление окнами
- **Help** - Справка

## 🔄 Сохранение конфигурации

### Автоматическое сохранение
- Когда: при закрытии GUI приложения
- Где: `config.ini` в корне проекта
- Формат: текстовый INI

### Содержимое config.ini
```ini
[VIDEO]
window_width=1280
window_height=720
fullscreen=false
vsync=true

[ASCII]
char_width=12
char_height=24

[CAMERA]
camera_speed=0.1
auto_rotate=true

[SCENE]
animation_speed=1.0
```

### Сброс параметров
```powershell
Remove-Item config.ini
.\run_gui.bat  # Загрузятся параметры по умолчанию
```

## ❓ Часто задаваемые вопросы

**Q: Как запустить приложение?**
A: `.\run_gui.bat`

**Q: Как управлять камерой?**
A: WASD для движения, стрелки для вращения, R для сброса, ESC для выхода

A: Автоматически при выходе из приложения

A: В файле `config.ini`

A: Удалите `config.ini` и перезапустите
